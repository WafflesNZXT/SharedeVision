# Sharede Vision Website

A modern, responsive website for Sharede Vision, a digital marketing agency specializing in website development, social media marketing, SEO, and Google Ads.

## Features

- Fully responsive design that works on all devices
- Modern gradient design with brand colors
- 7 pages: Home, About Us, Website Development, Social Media Marketing, Google Ads, SEO, and Contact Us
- Built with HTML5, CSS3, and JavaScript
- Fast loading and optimized for performance

## Pages

1. **Home** - Overview of services and company introduction
2. **About Us** - Company history, mission, and process
3. **Website Development** - Details about website development services
4. **Social Media Marketing** - Information about social media marketing services
5. **Google Ads** - Details about Google Ads (PPC) services
6. **SEO** - Information about SEO services
7. **Contact Us** - Contact form and information
   
## Setup

1. Clone the repository
2. Open index.html in your browser
3. No build process required - pure HTML, CSS, and JavaScript
